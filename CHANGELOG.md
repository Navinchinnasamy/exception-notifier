# Changelog

All notable changes to `exception-notifier` will be documented in this file.

## [1.0.0] - 2026-01-19

### Added
- Initial release
- Beautiful responsive HTML email template
- Zero-configuration installation for Laravel
- Simple setup for Lumen
- Rate limiting to prevent email spam
- Exception grouping by type and message
- Environment-specific configuration
- Mobile-friendly email design
- Secure exception filtering
- Artisan command for testing configuration
- Support for Laravel 5.3+ and Lumen 5.3+
- Comprehensive documentation

### Features
- Automatic exception catching and email sending
- Configurable recipient lists (TO and CC)
- Professional email template with exception details
- Request information in emails (URL, method, IP, user agent)
- Environment information (app name, environment, timestamp)
- Smart rate limiting per exception type
- Exception grouping to reduce email volume
- Filtering of common exceptions (404, validation, etc.)
- Support for multiple Laravel/Lumen versions

### Configuration
- Environment variable based configuration
- Sensible defaults for immediate use
- Advanced configuration options available
- Easy customization of email template
- Flexible exception filtering rules
